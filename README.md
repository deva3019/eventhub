# EventHub - College Event Management System

A modern, responsive college event management system built with Flask, MongoDB, and Tailwind CSS.

## Features

✅ **Modern UI** - Navy Blue theme with dark/light toggle
✅ **Twinkle Star Animations** - Beautiful animated background
✅ **Staff Authentication** - Secure login/registration
✅ **Event Management** - Create, edit, delete events
✅ **Event Registration** - Students can register for events
✅ **Responsive Design** - Works on all devices
✅ **Real-time Search** - Filter events instantly
✅ **Export Registrations** - CSV export functionality
✅ **MongoDB Integration** - Scalable database

## Technology Stack

- **Backend:** Flask 2.3+
- **Database:** MongoDB
- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Deployment:** Render.com
